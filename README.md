# 简历生成器

一个简洁易用的中文简历生成器，支持实时预览、多风格切换和本地数据存储。纯前端实现，无需任何外部依赖，打开即用。

## 功能

### 编辑功能
- **基本信息** — 姓名、职位、邮箱、电话、地点、个人网站
- **个人简介** — 自由文本编辑
- **工作经历** — 动态添加/删除，含公司、职位、时间、描述
- **教育背景** — 动态添加/删除，含学校、学位、时间
- **项目经历** — 动态添加/删除，含项目名称、时间、描述
- **技能** — 多行输入，每行一个类别（支持 `类别: 技能1, 技能2` 格式）
- **语言能力** — 多行输入，每行一种语言
- **证书** — 多行输入，每行一个证书
- **头像上传** — 支持图片上传，最大 2MB

### 样式功能
- **3 种布局风格**：
  - **经典** — 纵向排列，头像矩形右侧对齐，适合传统行业
  - **现代** — 彩色侧边栏 + 主内容区，信息层级清晰
  - **创新** — 左侧彩色竖条 + 时间线，前卫设计感
- **5 种颜色主题** — 蓝色、绿色、紫色、橙色、深色
- **模块显示控制** — 独立开关各模块的显示/隐藏

### 导出功能
- **导出 PDF** — 调用浏览器打印功能，已优化打印样式
- **导出 HTML** — 生成独立的 HTML 文件，含内联样式，可直接分享或打印

### 数据管理
- **本地存储** — 自动保存/恢复简历数据（localStorage）
- **示例数据** — 一键加载示例简历快速预览
- **清除数据** — 清除所有本地保存的内容

## 快速开始
### 在线
https://freadchips.github.io/my-resume/

### 本地
直接在浏览器中打开 `index.html` 即可使用

```bash
# 克隆或下载项目后
cd my-resume
# 直接打开
open index.html    # macOS
start index.html   # Windows
```

## 文件结构

```
my-resume/
├── index.html      # 主页面结构（编辑器 + 预览面板）
├── styles.css      # 样式文件（布局、主题、响应式、打印样式）
├── script.js       # 交互逻辑（数据管理、渲染、导出）
└── README.md       # 本文件
```

## 技术栈

- **前端** — 纯 HTML / CSS / JavaScript
- **无外部依赖** — 零框架，轻量级
- **响应式设计** — 支持桌面、平板、手机
- **打印优化** — 专门优化的 PDF 导出样式

## 浏览器兼容

支持现代浏览器（Chrome、Firefox、Safari、Edge）的打印功能。
